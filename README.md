# Jon Waaler's JavaScript Quiz Website

## Purpose

To learn how to use more advaced DOM using vanilla JS. Point system scoring.

- +20 for correct answer
- -10 for skipping
- -5 for wrong answer
- Bonus Points = time left \* 0.5
